## How to Take Screenshots

### Windows
- Press `Win + Shift + S` for Snipping Tool
- Or use Windows Game Bar: `Win + G`

### macOS
- Press `Cmd + Shift + 4` for selection screenshot
- Press `Cmd + Shift + 5` for recording

### Browser Extensions
- [Awesome Screenshot](https://www.awesomescreenshot.com/)
- [Fireshot](https://getfireshot.com/)

## Creating GIF Demos

1. **Record your screen** using OBS Studio, ScreenToGif, or browser extension
2. **Keep it short** (5-10 seconds max)
3. **Optimize file size** using [ezgif.com](https://ezgif.com/optimize)
4. **Target size**: Under 5MB for fast loading

## Tips for Great Screenshots

✅ **DO:**
- Use a clean browser without extensions visible
- Show real, interesting data (not placeholder text)
- Use good contrast and readable text
- Crop to focus on relevant content
- Use consistent styling across all screenshots

❌ **DON'T:**
- Include personal information or API keys
- Use low resolution or blurry images
- Include browser UI unless necessary
- Use dark mode if your app is light (or vice versa)
